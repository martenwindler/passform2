# passform2

ROS2 ecosystem managing RFID events, heartbeats, system states, Raspberry Pi clusters.

```text
passform2_ws/
├── src/
│   ├── passform2/                # Ros2 package knowledge update
│   ├── passform_agent_backend/   # Backend
│   ├── passform_agent_frontend/  # Frontend
│   └── passform2_rfid/           # RFID
└──                               # Other
```
Build.
```text
make build
```

Build and start directly.
```text
make all
```

Start, when already built.
```text
make run
```
